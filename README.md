# Django Expense Tracker

A simple web application built using **Django** to track daily expenses. Users can add, view, and delete expenses with a clean UI.

## Features

* Add daily expenses
* View all expenses in a table
* Delete expenses
* Decimal (float) amount support

## Tech Stack

* Python
* Django
* SQLite
* HTML & CSS
